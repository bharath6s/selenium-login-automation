 # selenium-login-automation
Beginner Selenium automation project using Java

## Description
Automated login functionality of OrangeHRM demo application using Selenium WebDriver.

## Scenarios Covered
- Valid login with correct credentials
- Invalid login with incorrect credentials

## Validations
- Dashboard URL verification for successful login
- Error message validation for failed login

## Tools & Technologies
- Java
- Selenium WebDriver
- Maven
- Chrome Browser

## How to Run
1. Clone the repository
2. Open the project in Eclipse
3. Run LoginTest.java as Java Application
